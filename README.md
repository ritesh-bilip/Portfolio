Exciting Project Launch

I am really happy to announce that my Portfolio Website is now live and hosted on Render: https://portfolio-site-7upw.onrender.com/.

My Portfolio Website is more than a showcase. It is a fully dynamic Django application designed to highlight my journey as a full stack developer.

My Portfolio Website project is about my journey as a stack developer and I have built it using Django.

From the panel to the PostgreSQL integration every piece of my Portfolio Website reflects my focus on building strong and reliable systems.

The deployment pipeline automates everything: it installs dependencies collects files runs migrations and serves the app with Gunicorn. This makes updates really smooth with every push.

By hosting my Portfolio Website on Render I have used modern cloud workflows that make deployment easy and professional.

My Portfolio Website is not a window into my projects but also a demonstration of my ability to design, debug and deliver scalable solutions from start to finish.

Check out my Portfolio Website explore my work and let me know what you think. I am excited to keep pushing boundaries and building more powerful applications.

Tech Stack

- Language: Python with the Django Framework

- Server: Gunicorn

- Deployment: Render Cloud Platform

- Version Control: Git and GitHub

Database

- PostgreSQL managed by Render

- Integrated via dj_database_url for environment variable configuration

- Automated migrations during deployment

Frontend

- HTML5, CSS3, JavaScript

- Dynamic templates powered by Django’s template engine

- Responsive design for desktop and mobile

Backend

- Django Framework for strong MVC architecture

- Admin Panel for managing my Portfolio Website content dynamically

- Authentication Ready for future user login integration
